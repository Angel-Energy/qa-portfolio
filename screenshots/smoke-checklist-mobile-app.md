# Smoke Checklist — мобильное приложение доставки / магазина

## Цель
Проверить, что после сборки или обновления
основные пользовательские сценарии работают
и приложение пригодно для дальнейшего тестирования.

## Область проверки
- запуск приложения
- авторизация
- навигация по основным разделам
- поиск товара
- работа карточки товара
- добавление в корзину
- оформление заказа

## Критерий прохождения smoke-теста
Smoke-тест считается успешным, если работают
все основные пользовательские сценарии:
- запуск приложения
- авторизация
- поиск
- открытие карточки товара
- добавление в корзину
- переход к оформлению заказа
